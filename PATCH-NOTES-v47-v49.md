## 🃏 v47 — 실제 캐쉬게임 AI 톤 (사람처럼 플레이)

### 문제였던 것
이전엔 펍에서도 AI가 4-bet/5-bet을 막 질러댔어. 강남, 마카오, 벨라지오 다 비슷한 톤. 실제 오프라인 캐쉬에선 안 일어나는 일.

### 바뀐 것
- **베뉴별 성향 분포 완전 재설계** — 펍은 콜링 스테이션 50%, 트라이튼은 PRO/GTO만
- **프리플랍 4-bet/5-bet 임계값 강화** — 3-bet 75+, 4-bet 87+, 5-bet 93+ 강도 필요
- **프리플랍 블러프 60% 감소** (4-bet+은 추가 30% 감소) — 실제 포커는 프리플랍 블러프 거의 없음
- **포지션 가중치 강화** — UTG 0.78× / BTN 1.20× → 얼리 포지션 매우 타이트, 버튼은 공격적

### 베뉴별 표준 분포

| 베뉴 | 주요 성향 |
|---|---|
| 펍 (동네) | 콜링 50% + 패시브 30% (4-bet 거의 없음) |
| 강남 | TAG 40% + 콜링 20% (한국 캐주얼) |
| 마카오 | TAG 40% + GTO 25% (아시아 강자) |
| 벨라지오 | PRO/GTO 60% (정통 톤) |
| WSOP | PRO 45% + GTO 30% (월드클래스) |
| 트라이튼 | PRO 60% + GTO 35% (거의 솔버) |

## 🔍 v48 — 통찰력 기반 점진 공개 + 이벤트 몰입

### 1) AI 성향, 더 이상 공짜로 안 알려줌

이전: AI 머리 위에 `🎯 TAG`, `🗿 ROCK` 다 보였음 → 너무 쉬웠음.

이제: **`통찰력(insight)` 능력**으로 핸드를 누적 관찰해야 성향이 단계별로 공개됨.

#### 4단계 공개 시스템

| 단계 | 표시 | 통찰력 0 | 통찰력 80 |
|---|---|---|---|
| 0 | `❓ 5/15 핸드 관찰 중` | 0~14핸드 | 0~5핸드 |
| 1 | `📊 타이트` / `📊 와이드` | 15핸드~ | 6핸드~ |
| 2 | `🔥 LAG (와이드/공격)` 4분면 | 30핸드~ | 14핸드~ |
| 3 | `🎯 TAG` 정확한 라벨 | 60핸드~ | 28핸드~ |

좌석을 클릭하면 **실제 포커 HUD 스타일** 스탯이 떠 — VPIP, PFR, 3-bet%, AF (포스트플랍 공격성). 통찰력 레벨이 높을수록 빨리 공개.

**예외**: Phil Ivey, Hellmuth 같은 유명 챔피언은 항상 공개 — 평판이 이미 있는 사람들이니까.

### 2) 특별 이벤트 발동 시 몰입형 정지

로열 플러시, 4 of a Kind, A 포함 너트 플러시, 100BB+ 빅팟, 7연승, 윤서아 첫 만남 등 **특별 모먼트**가 나오면:

- 화면 전체 라디얼 dim (중앙은 밝고 가장자리만 어둡게)
- 액션 패널·플레이어 카드 채도 60%로 떨어짐
- **BGM 8초간 페이드다운** (볼륨 20%)
- 모달 버튼 펄스 애니메이션 — 의식적으로 클릭하게
- 다음 핸드는 모달 닫을 때까지 절대 안 시작

### 3) 카드 탭 버그 픽스

이전: 카드 뒷면 탭 → 프로필 모달이 같이 열리는 버그.
원인: 이벤트 버블링.
픽스: `event.stopPropagation()` 박음.

## 🎓 v49 — GTO 솔버 + AI 코치 + 주간 리포트

### 1) GTO 솔버 통합

**9-handed 캐쉬게임 표준 GTO 차트** 내장:

- **포지션별 오픈 레인지** — UTG 14%, MP 18%, CO 28%, BTN 45%, SB 35%
- **3-bet 레인지** — UTG 5%, BTN 13%, BB 17%
- **베팅 사이징 추천** — 오픈 2.5BB, 3-bet 9BB(IP)/11BB(OOP), 4-bet 24BB
- **포스트플랍 보드 텍스처 분석** — 드라이 보드 33% C-bet, 웻 보드 75% 폴라
- **팟 오즈 자동 계산** — 콜/폴드 판단

### 2) AI 코치 모드 (행동 후)

**매 액션마다 우상단 토스트로 GTO 비교 피드백:**

```
🎓 AI 코치
✓ 정확     GTO: 오픈 레이즈 (2.5BB)
BTN 오픈 45% 레인지 포함
```

색깔 코딩:
- 🟢 **✓ 정확** — GTO와 액션 일치
- 🟡 **△ 무난** — 약간 다르지만 큰 손해 아님
- 🔴 **✗ 더 좋은 선택** — GTO 추천과 어긋남

**왜 액션 후?** — 학습 효과는 본인이 결정한 다음 비교해야 가장 크고, 실시간 가이드는 게임 재미를 깎음.

토글: 설정 메뉴 → "🎓 AI 코치 모드" 체크박스로 ON/OFF.

### 3) 주간 약점 리포트

로비 → **📊 통계 & 분석 → 주간 약점 리포트**

최근 7일 결정 데이터를 분석해서:

- **전체 점수** (100점 만점) + **A/B/C/D/F 등급**
- **전주 대비 추세** (▲/▼)
- **스트릿별 점수 바** — 프리플랍/플랍/턴/리버 각각
- **가장 약한 스트릿 자동 식별** + 학습 팁

예시 출력:
> Overall: **68/100 (B 등급)** ▲ +4 (전주 대비)
>
> 프리플랍 82점 · 플랍 71점 · **턴 52점** ⚠️ · 리버 67점
>
> 💡 턴 — 베럴 또는 체크?
> 플랍 C-bet 후 턴은 결정적. 페어/드로우 있으면 베럴 컨티뉴, 에어/약한 페어는 체크-콜 또는 자이브업.

데이터가 10개 미만이면 "더 플레이하세요" 안내.

## 💬 플레이 팁

1. **새 베뉴 진입 시 첫 15~30핸드는 관찰 모드** — 통찰력 낮으면 더 오래. 옆 사람 4분면 (TIGHT/WIDE × PASSIVE/AGGRO) 파악되기 전까진 보수적으로.
2. **펍에서 4-bet 만나면 진짜로 너트** — 펍 4-bet은 거의 안 일어남. 보이면 AA-KK 확정.
3. **코치 모드 ON으로 100핸드 → 주간 리포트 → 약한 스트릿 집중 학습** — 이게 가장 빠른 실력 상승 루트.
4. **트라이튼은 GTO 솔버 톤 60%+** — 익스플로잇 불가. 본인도 GTO에 가깝게 플레이해야 함.
