# 🎰 NEON HOLDEM

> 인디 텍사스 홀덤 RPG 캐리어 게임. 동네 펍에서 트라이튼 하이스테이크까지.

[**🎮 게임 플레이하기 →**](https://rlawnaud3-arch.github.io/neon-holdem/)

---

## ✨ 특징

### 🎯 캐리어 RPG 시스템
- 6개 베뉴 단계별 진행: 동네 펍 → 강남 → 마카오 → 라스베가스 → WSOP → 트라이튼
- 시작 자금 $500에서 $5,000,000 목표
- 자금 도달 시 자동 베뉴 해금

### 🎮 능력 시스템 (악마의 열매식)
- **8 기본 능력**: 통찰력, 포커페이스, 블러프, 익스플로잇, 멘탈, 인내심, 수학, 압박
- **15 특수 능력**: Phil Ivey, Daniel Negreanu, 김연수 등 전설 플레이어 격파 시 시그니처 능력 흡수
- 액션마다 XP 자동 누적, 레벨 1→100

### 📊 프로급 학습 도구
- 실시간 트래커: VPIP, PFR, 3-bet, C-bet, WTSD, W$SD, AF, BB/100
- 42문제 GTO 퀴즈 (4 카테고리)
- 13×13 핸드 매트릭스 (5 포지션)
- 의사결정 자동 채점 (✓/△/✗)
- 약점 분석 + 학습 가이드

### 🎒 컬렉션 시스템
- 41개 아이템, 4 등급 (일반/레어/영웅/전설)
- 12 슬롯: 모자, 선글라스, 마스크, 재킷, 카드덱 등
- 12 캐릭터 아바타
- 5종 카드 뒷면

### 🎯 챌린지 & 보스
- 일일 챌린지 (자정 갱신)
- 베뉴별 보스 핸드
- 시즌별 보상

### 🎷 사운드
- 3개 BGM 트랙: Funky Casino / Lo-Fi Jazz / 미니멀 피아노
- Tone.js 기반 실시간 신디사이즈
- 15+ SFX (카드 딜, 칩, 폴드, 레이즈, 올인, 승리 등)

## 🎮 플레이 방법

### 기본 액션
| 키 | 액션 |
|---|---|
| `F` | 폴드 |
| `C` | 콜 / 체크 |
| `R` | 레이즈 |
| `A` | 올인 |

### 진행 흐름
1. 첫 진입 시 슬롯 1 자동 선택, 시작 자금 $500
2. 펍 ($1/$2) 입장 → 핸드 플레이
3. 자금 모으면 다음 베뉴 자동 해금
4. 유명 플레이어 만나면 격파 → 능력 흡수
5. 일일 챌린지 수행 → 추가 보상
